# Vox — голосовой ввод для Windows

> Зажми **правый Ctrl**, говори, отпусти — и текст появится прямо там, где курсор.

В любом окне: чат, браузер, блокнот, поле поиска, терминал, IDE, CLI. Всё работает офлайн, на твоём компьютере, без интернета и без отправки голоса куда-либо.

---

## 🎯 Зачем это нужно

- ✍️ **Печатаешь много текста.** Сообщения, документы, заметки — теперь голосом, в 3-4 раза быстрее.
- 💬 **Болят запястья.** Меньше клавиатуры — меньше нагрузки на руки.
- 🌍 **Любое окно — без копипаста.** Не нужно переключаться в специальное приложение и копировать текст обратно.
- 🔒 **Приватность.** Распознавание происходит у тебя на компьютере. Голос никуда не уходит.
- 🇷🇺 **Понимает русский.** И автоматически превращает айтишный сленг в нормальные английские термины: «реакт» → React, «гитхаб» → GitHub, «юзстейт» → useState.

## 🚀 Установка за минуту

### 1. [Скачай установщик](https://github.com/privatekey7/Vox_1.0.0/releases/tag/v1.0.0)

Запусти `vox-setup.exe` (≈29 МБ).

### 2. Пройди мастер установки

1. Жми **Далее**.
2. Поставь галочки по желанию:
   - 🖥️ Создать ярлык на рабочем столе
   - ⚡ Запускать Vox при старте Windows *(рекомендуем)*
3. Жми **Установить**, подтверди UAC.
4. Оставь галочку **«Запустить Vox»** и жми **Завершить**.

### 3. Подожди модель *(только в первый раз!)*

При первом запуске Vox скачает модель распознавания речи (≈45 МБ). Это занимает 30 секунд — зависит от интернета.

### 4. Готово 🎉

В трее (рядом с часами) появится иконка Vox — значит всё работает.

---

## 🎙 Как пользоваться

1. Открой любое окно, где хочешь что-то напечатать.
2. **Кликни мышкой** в поле для ввода.
3. **Зажми правый Ctrl** — услышишь короткий «бип».
4. **Говори** — спокойно, как с человеком.
5. **Отпусти Ctrl** — через секунду текст окажется в поле.

> Если автовставка не сработала (некоторые приложения её блокируют) — текст всегда в буфере обмена, просто нажми **Ctrl+V**.

| Скажи | Получится |

Vox знает больше тысячи айтишных терминов — React, Python, Docker, Kubernetes, AWS, и так далее.

## 🔧 Меню в трее

Правый клик на иконке Vox:

| Пункт | Что делает |
|-------|------------|
| **Клавиша: …** | Текущая горячая клавиша (обновляется автоматически) |
| **Сменить клавишу ▸** | Подменю для выбора другой клавиши: Правый/Левый Ctrl, Правый/Левый Shift, Правый/Левый Alt, Caps Lock, Scroll Lock, Pause, F1–F12 |
| **Открыть логи** | Открывает папку с логами |
| **Открыть папку с моделями** | Открывает папку с моделями |
| **Выход** | Закрывает Vox |

> По умолчанию горячая клавиша — **правый Ctrl**. Можно сменить через подменю **Сменить клавишу ▸** в трее: выбор сохраняется в `%APPDATA%\vox\config.json` и применяется сразу, без перезапуска.

## ❓ Если что-то идёт не так

<details>
<summary><b>Горячая клавиша не работает</b></summary>

Закрой Vox через трей (правый клик → **Выход**) и запусти от имени администратора:
правый клик на ярлыке → **«Запуск от имени администратора»**.

> Windows требует прав администратора, чтобы программа могла перехватывать правый Ctrl на системном уровне.
</details>

<details>
<summary><b>Vox не слышит голос</b></summary>

1. Проверь микрофон: **Параметры Windows** → **Звук** → проверь уровень микрофона.
2. Говори громче и чётче, ближе к микрофону.
3. Открой логи через трей-меню (**Открыть логи**) — там подробности.
</details>

<details>
<summary><b>Иконка в трее не появилась</b></summary>

1. Открой `%APPDATA%\vox\logs\vox.log` (можно вставить эту строку в адресную строку Проводника).
2. Посмотри последние строки — там будет ошибка.
3. Если непонятно — приходи с этим логом за помощью.
</details>

<details>
<summary><b>Текст не вставляется автоматически</b></summary>

Некоторые приложения блокируют автовставку. Текст всё равно в буфере обмена — просто нажми **Ctrl+V** вручную.
</details>

<details>
<summary><b>Защитник Windows ругается на установщик</b></summary>

В окне SmartScreen нажми **«Подробнее»** → **«Выполнить в любом случае»**.

Установщик пока не подписан сертификатом, поэтому Windows подозрительно к нему относится. Это нормально для open-source программ.
</details>

## 📁 Где Vox хранит файлы

| Где | Что |
|-----|-----|
| `C:\Program Files\Vox\` | Сама программа |
| `%APPDATA%\vox\models\` | Модель распознавания (≈70 МБ) |
| `%APPDATA%\vox\logs\` | Логи работы |

Чтобы быстро открыть `%APPDATA%\vox\` — нажми **Win+R**, введи `%APPDATA%\vox` и Enter.

---

## ⚡ Автозапуск при старте Windows

Самый простой способ — поставить галочку **«Запускать Vox при старте Windows»** прямо в установщике.

Если забыл — переустанавливать не нужно:

1. **Win+R** → `shell:startup` → Enter.
2. Перетащи туда ярлык Vox с рабочего стола.
3. Правый клик на ярлыке → **Свойства** → **Дополнительно** → галочка **«Запуск от имени администратора»**.

---

## 🗑 Как удалить

**Параметры Windows** → **Приложения** → найди **Vox** → **Удалить**.

Папки `%APPDATA%\vox\models` и `%APPDATA%\vox\logs` не удаляются автоматически — если они тебе не нужны, удали вручную.
